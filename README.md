# Mundo Animal – Página Responsiva

## Descrição

Página web que apresenta informações sobre diferentes animais, curiosidades e a biodiversidade do planeta. Desenvolvida com **HTML5, CSS3, Flexbox e Grid**, a página é **responsiva**, funcionando bem em celulares, tablets e desktops.

## Estrutura do Projeto

```
mundo-animal/
│
├── index.html      # Página principal
├── style.css       # Arquivo de estilos
└── README.md       # Este arquivo
```

## Funcionalidades

* **Cabeçalho** com título e subtítulo.
* **Seção de destaque** sobre biodiversidade em risco.
* **Cards principais** com informações sobre leões, elefantes e curiosidades do mundo animal.
* **Galeria extra** com mais animais.
* **Rodapé** com informações do autor e tecnologias usadas.

## Tecnologias

* HTML5
* CSS3 (Flexbox e Grid)
* Google Fonts
* Media Queries (responsividade)

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Navegue pelas seções para conhecer os animais e curiosidades.
